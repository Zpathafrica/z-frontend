# ZPath

A WhatsApp guide into Zcash privacy. Built for East Africa.

[zpath.africa](https://zpath.africa) &nbsp;·&nbsp; [Zcash Forum](https://forum.zcashcommunity.com) &nbsp;·&nbsp; [MIT License](LICENSE)

---

ZPath takes anyone from zero knowledge to their first shielded Zcash transaction through a simple WhatsApp conversation. No app. No account. English and Kiswahili.

**Layer 1 · Learn** &nbsp; What Zcash is, why shielded transactions are private, how to set up a wallet safely.

**Layer 2 · Acquire** &nbsp; How to buy ZEC with M-Pesa, step by step. Or practice free on testnet first.

**Layer 3 · Verify** &nbsp; Make a shielded transaction. Paste the transaction ID. Confirmed on-chain.
